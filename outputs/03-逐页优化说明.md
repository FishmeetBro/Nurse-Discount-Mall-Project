# 五页面优化说明与完整代码索引

> 完整 WXML / WXSS / JS / JSON 均已直接写入项目，以下路径即最终代码。为避免文档副本与运行代码不同步，以这些源文件为唯一权威版本。

## 1. 首页

代码：[WXML](../pages/home/home.wxml) · [WXSS](../pages/home/home.wxss) · [JS](../pages/home/home.js) · [JSON](../pages/home/home.json)

- 使用 1200×600 本地 ImageGen Banner，消除原纯样式占位画面。
- 商品列表改用公共 `product-card`，双价格、资质、加购状态统一。
- 商品/认证/购物车状态接入加固缓存模块。
- 首屏保留静态骨架，缓存异常不会阻断页面渲染。
- 保留卡片按压和分类切换反馈；根页面不使用透明度首帧动画，避免开发者工具热重载停帧导致白屏。

## 页面公共调整

- 五页全部使用公共 `compliance-notice`。
- 全局加入 180ms 按压反馈；弹层使用 200–240ms 动画，根页面始终强制可见。
- 布局单位为 rpx / 百分比 / flex；不存在页面固定 px 布局。
- 不使用网络图片、第三方组件、云开发、支付或物流接口。
