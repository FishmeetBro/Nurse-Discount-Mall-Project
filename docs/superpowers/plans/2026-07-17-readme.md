# 中文项目 README Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 创建并上传一份准确、专业、适合 GitHub 首页展示的中文项目 README。

**Architecture:** README 作为仓库根目录的单一项目入口，内容直接引用当前源码、目录和验证脚本，不增加运行依赖。使用仓库内 Banner 图片作为视觉预览，并明确纯本地 Demo 与合规边界。

**Tech Stack:** GitHub Flavored Markdown、微信小程序原生 WXML/WXSS/JS、Git。

## Global Constraints

- 全文使用简体中文。
- 不写入虚假的线上地址、构建状态、下载量或真机测试结论。
- 明确所有支付、订单、物流及佣金数据均为本地模拟数据。
- 明确仅支持二级分佣、禁止医院内推销、不得夸大医疗器械疗效。
